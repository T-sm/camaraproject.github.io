API Architecture

Abstraction from Network APIs to Service APIs is necessary:

  • To hide telco complexity making APIs easy to consume for customers with no telco expertise (user-friendly APIs)
  • To fulfil data privacy and regulatory requirements
  • To facilitate application to network integration
